Tribute page

The objective of this challenge is to build a tribute page similar to the tribute page from freeCodeCamp.

This involves a Title (name of the great personage), a picture, a list describing events or facts and a link for more information.

  • Should conform to the user stories.
  • Written by me in codepen.io (later run locally in docker nginx).

The overall structure should look something like this:

            main
              title
              img
              list
                ...facts
              link
            /main

Survey Form

The objective of this challenge is to build a survey form similar to the survey form from freeCodeCamp.

This involves a Title (subject), a short description and a set of form fields in a form with some validation.

  • Should conform to the user stories.
  • Written by me in codepen.io (later run locally in docker nginx).

The overall structure should look something like this:

            main
              h1
              p
              form
                label/text
                label/email
                label/number
                label/dropdown
                fieldset
                  checkboxes
                fieldset
                  radios
                label/textarea
                submit
            /main

Technical Documentation Page

The objective of this challenge is to build a technical document page similar to the technical document page by freeCodeCamp.

It involves having a navigation (fixed to the left on regular sized devices) and should cover at least 5 topics. I have chosen to document my work on the 5 pages built for the Responsive Web Design challenge.

  • Should conform to the user stories.
  • Written by me in codepen.io (later run locally in docker nginx).

The overall structure should look something like this:

          wrapper
            nav
              heading
              links 1-5
            /nav
            main
              sections 1-5
                heading
                p (2)
                li (2)
                code
            /main
          /wrapper
